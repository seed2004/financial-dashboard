7 How To Reuse This Template

Apply this to a different company in four steps

  1. Pull the source data. Go to the company's 10-K / 10-Q on SEC EDGAR (free, primary source) or a financial data site like stockanalysis.com for the income statement, balance sheet, cash flow statement and ratios.
  2. Edit the COMPANIES object near the top of the <script> section of this file. Copy the AAPL block, rename the key to the new ticker, and replace each array of six numbers (FY-4 … FY-1, FY0, TTM) with the new company's figures.
  3. Peers are automatic. The getPeers() function picks 5 comparison companies from everything else in COMPANIES (matching industry first, then sector) — no manual peer list to maintain. Add your own SCORECARD with 1–5 ratings and rationale for each of the five categories.
  4. Add an <option> to the company dropdown in the HTML and reload the file in your browser — every chart, table and the DCF will recalculate automatically.
